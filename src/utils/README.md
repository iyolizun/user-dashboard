# User Dashboard README

# Project Overview
This is a user dashboard project built with Node.js and Express.js. It provides a simple and intuitive interface for users to manage their data.

# Prerequisites
- Node.js (>= 14.x)
- npm (>= 6.x)
- Express.js (>= 4.x)
- MongoDB (>= 4.x)

# Installation
```bash
npm install
```

# Running the Application
```bash
node app.js
```

# API Endpoints
- GET /users: Retrieves a list of all users
- GET /users/:id: Retrieves a specific user by ID
- POST /users: Creates a new user
- PUT /users/:id: Updates a specific user
- DELETE /users/:id: Deletes a specific user

# API Documentation
The API documentation is available in the `docs` directory.
